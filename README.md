<h2>Overview</h2>
<p>Portfolio is a powerful developer tool that enables the creation of visually stunning 3D portfolio applications using React and Vite.</p>
<p><strong>Why Portfolio?</strong></p>
<p>This project empowers developers to build engaging user experiences with modern web technologies. The core features include:</p>
<ul class="list-disc pl-4 my-0">
<li class="my-0">🎨 <strong>3D Rendering:</strong> Create immersive experiences with Three.js, enhancing user engagement.</li>
<li class="my-0">📱 <strong>Responsive Design:</strong> Leverage Vite and Tailwind CSS for seamless styling across devices.</li>
<li class="my-0">🔄 <strong>Dynamic Components:</strong> Utilize reusable components for easy maintenance and modularity.</li>
<li class="my-0">✨ <strong>Animations with GSAP:</strong> Incorporate smooth transitions to captivate users and improve interaction.</li>
<li class="my-0">✅ <strong>Code Quality Assurance:</strong> Enforce best practices with ESLint, ensuring a robust codebase.</li>
<li class="my-0">📬 <strong>Integrated Contact Form Real Time:</strong> Facilitate user inquiries and enhance communication effortlessly.</li>
</ul>

<h2>Getting Started</h2>
<h3>Prerequisites</h3>
<p>This project requires the following dependencies:</p>
<ul class="list-disc pl-4 my-0">
<li class="my-0"><strong>Programming Language:</strong> JavaScript</li>
<li class="my-0"><strong>Package Manager:</strong> Npm</li>
</ul>
<h3>Installation</h3>
<p>Build portfolio from the source and intsall dependencies:</p>
<ol>
<li class="my-0">
<p><strong>Clone the repository:</strong></p>
<pre><code class="language-sh">❯ git clone https://github.com/namngxthanh/portfolio
</code></pre>
</li>
<li class="my-0">
<p><strong>Navigate to the project directory:</strong></p>
<pre><code class="language-sh">❯ cd portfolio
</code></pre>
</li>
<li class="my-0">
<p><strong>Install the dependencies:</strong></p>
</li>
</ol>
<p><strong>Using <a href="https://www.npmjs.com/">npm</a>:</strong></p>
<pre><code class="language-sh">❯ npm install
</code></pre>
<h3>Usage</h3>
<p>Run the project with:</p>
<p><strong>Using <a href="https://www.npmjs.com/">npm</a>:</strong></p>
<pre><code class="language-sh">npm start
</code></pre>
